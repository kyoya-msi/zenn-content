## はじめに


## エラー事例
SpringBoot+Thymeleafのアプリケーションにて、入力フォームの画面を表示するためにGETリクエストを実行すると以下のエラーが発生。
```text
Caused by: java.lang.IllegalStateException: Neither BindingResult nor plain target object for bean name 'exampleForm' available as request attribute
```

## 原因
描画対象のHTMLファイルにFormインスタンスを参照するThymeleaf属性を付与しているが、Controllerから渡されたModelにFormインスタンスが格納されていない状態となっていること。

## 対処法
Controllerの初期表示メソッドで、空のFormインスタンスを生成してSpringレイヤーに渡すことが必要。
以下の3パターンで実現できる。

## パターン1:Modelを使って明示的に渡す
引数にModelを受け取り、addAttributeメソッドでインスタンス化したFormインスタンスを渡す。
```java
@Controller
@RequestMapping( "/exam" )
public class Example
{
	@GetMapping( "" )
	public String index( Model model )
	{
		// "exampleForm"という名前で、空のFormインスタンスをModelに先頭アドレスを格納する。
		model.addAttribute( "exampleForm", new ExampleForm() ) ;
		return "input-form" ;
	}
}
```

## パターン2:メソッド引数にFormクラスを追加する
引数にFormインスタンスを追加する。
HandlerAdapterがメソッド実行時にインスタンス化してModelに先頭アドレスを格納する。
```java
@Controller
@RequestMapping( "/exam" )
public class Example
{
	@GetMapping( "" )
	public String index( ExampleForm form )
	{
		return "input-form" ;
	}
}
```

## パターン3:@ModelAttributeアノテーションを使用する
ModelAttributeアノテーションを付与したメソッド内でFormインスタンスを生成およびHandlerAdapterへ返却する。
※注意:publicメソッドにすること。
```java
@Controller
@RequestMapping( "/exam" )
public class Example
{
	@ModelAttribute
	public ExampleForm setUpForm()
	{
		return new ExampleForm() ;
	}

	@GetMapping( "" )
	public String index()
	{
		return "input-form" ;
	}
}
```
